# Cursor Landing Page

This project is a recreation of the Cursor landing page using HTML and CSS.

---

## Sections Recreated

- Navigation Bar  
  - Logo  
  - Features, Enterprise, Pricing, Resources links  
  - Sign in and Download buttons  

- Hero Section  
  - Main headline text  
  - Call-to-action button (Download for macOS)

- Preview Section  
  - Screenshot image of the Cursor application  

---

## Fonts and Colors Used

### Fonts
- Default system font (sans-serif)

### Colors
- Dark gradient background (black / dark brown tones)
- White and light gray text
- White download button with dark text
