# Cursor Landing Page

This project is a recreation of the Cursor landing page using HTML and CSS.

## Sections Recreated
- Navigation Bar (Logo, links, Sign in & Download buttons)
- Hero Section (headline and CTA button)
- Preview Section (application screenshot)

## Fonts and Colors Used
**Fonts**
- Default system font (sans-serif)

**Colors**
- Dark gradient background (black / dark brown)
- White and light gray text
- White buttons with dark text
